## AAD_Phase_2_Practice_Project
A simple project to display names of top interns in Google Africa Developer Scholarship (GADS).

This **GADS 2020 Leaderboard.** displays top notchers which are categorised into Skills and Hours spent learning.
There are **Top 20 learners in the Learning Leaders’** category and **Top 20 learners in the Skill IQ Leaders’** category.

# Library used:
1. Retrofit
2. Databinding
3. Moshi
4. Glide
5. Room
6. Couroutine
7. Lifecycle Extensions
8. ViewPager

## Dependencies can be found here
- [ViewPager](https://developer.android.com/jetpack/androidx/releases/viewpager2)
- [Navigation Component](https://developer.android.com/jetpack/androidx/releases/navigation)
- [Retrofit](https://github.com/square/retrofit)
- [Glide](https://github.com/bumptech/glide)

## SDK Requirements used
- Minimum SDK Requirement - android API 22
- Target SDK - android API 30
